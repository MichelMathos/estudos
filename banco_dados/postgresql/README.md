# **Sobre**
Este projeto se trata de um agendador de tarefas simples, do qual é possível realizar as tarefas de um CRUD.
Ao iniciar, o sistema irá apresentar no topo as tarefas agendas, e logo abaixo o menu das opções.

O agendador é composto por 4 campos distintos:
- ID: autoincremental. Também responsvel por fazer a seleção das tarefas em caso de edição/atualização das tarefas e de deletá-las.
- Descrição: que trata da tarefa em si, o que há de ser feito.
- Data: para qual dia a tarefa deverá ser realizada.
- Horário: em qual horário especifico a tarefa deverá ser realizada

## Linguagem
A linguagem utilizada é Ruby

## Banco de dados
O banco de dados é o PosgreSQL
